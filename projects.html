Projects

A list of my academic and personal projects, organized by category and in reverse chronological order.

Coding

  • Genuary 2021 (January 2021): Generative art created as part of the #genuary2021 challenge. Completed 27/31 prompts using p5.js. View my Genuary 2021 art.

  • How to Start Making Money Blogging (June 2020): An idle game about blogging, based on a Wikihow article. Ranked 5th in the Random Wikihow Jam. Play on itch.io.

  • Tweet Tweet Snake (May 2020): A version of the classic snake game written in under 560 characters of code, using TIC-80 and Lua. Submitted to TweetTweetJam 4. Play on itch.io.

  • Football Head Impacts Study (Fall 2019): A data science independent study project using Python to investigate head impact data collected from Lake Forest College’s football team. This project was part of the Applied Mathematics and Advanced Computation Lab at Lake Forest College. Originally scheduled to be presented at the Lake Forest College 2020 Steven Galovich Student Symposium (cancelled due to COVID-19).

  • Rush Behavioral Commitment Project (Fall 2019): Worked with a researcher from Rush University Medical Center to develop a SuperLab experiment for testing different behavioral commitment strategies.

  • Twitter Bots (2018 - 2019): Assorted Twitter bots created using Tracery and Cheap Bots Done Quick. Bots include @unicode_faces, @XFilesPlotBot, and @cs_club_img_bot. Used for teaching Tracery in the LFC Computer Science Club.

  • Video Store Database Website (Fall 2018): A web interface for a hypothetical video store, built using SQL and PHP as a group project for a class on database systems. Github Repo.

  • Arcade Game (2017 - 2018): A one- or two-player shooter game made in Java, which was part of an arcade machine that CS Club built and programmed from scratch. Github Repo.

  • Richter Scholar Project (Summer 2017): A research project using Python to analyze a large data set of news stories to discover relationships between news sentiment and stock prices. Github Repo.

Writing

  • Ni Narrador Ni Narradora: La Traducción de Género Ambiguo en Escrito en el Cuerpo (Spring 2018): A paper analyzing strategies for translating ambiguous or nonbinary gender between English and Spanish, taking as a case study Jeanette Winterson’s novel Written on the Body and its Spanish translation by Encarna Castejón. Written in Spanish. Published in Intertext, Lake Forest College’s undergraduate humanities journal.
